Frequently asked questions.

Does Sidebar Buddy touch, move, or modify my files?

No. Sidebar Buddy is a shortcut manager — it only stores the path to your folders, never the folders themselves. It reads folder names to display them in the sidebar and opens Windows Explorer to the correct location when you click one. Your files are never moved, renamed, copied, or deleted by Sidebar Buddy under any circumstances.

Is it safe to try? Does it change any Windows settings?

Yes, completely safe. There is no installer — the app runs as a single .exe. The only optional system change is "Launch on startup," which you control in Settings. Uninstalling is as simple as deleting the file.

What happens when the 15-day trial ends?

The app will prompt you to activate your license before it launches. Your folder list and settings are preserved — nothing is deleted. Enter your license key to pick up right where you left off.

Does it work with network drives, USB drives, or OneDrive?

Yes. Any folder Windows can see — local drives, mapped network drives, USB, OneDrive, Dropbox — works as a bookmark. If the drive is offline when you click it, Windows Explorer handles the error the same way it always would.

Do I need admin rights to run it?

No. Sidebar Buddy runs as a standard user application and requires no elevated permissions.

Is there any telemetry, analytics, or cloud sync?

None. Your folder list lives in a single JSON file on your machine. Nothing is ever transmitted to any server. The only internet connection the app makes is during license activation — and that's a one-time check.

Which versions of Windows are supported?

Windows 10 (version 1903 or later) and Windows 11. Both 32-bit and 64-bit are supported. It requires .NET 8, which the installer bundles — no separate download needed.
